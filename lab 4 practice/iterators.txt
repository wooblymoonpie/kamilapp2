### üîß **–ü—Ä–∏–º–µ—Ä: —Å–≤–æ–π –∏—Ç–µ—Ä–∞—Ç–æ—Ä –æ—Ç 1 –¥–æ 5**

```python
class CountToFive:
    def __init__(self):
        self.num = 1

    def __iter__(self):
        return self

    def __next__(self):
        if self.num <= 5:
            val = self.num
            self.num += 1
            return val
        else:
            raise StopIteration

# –ò—Å–ø–æ–ª—å–∑—É–µ–º –∏—Ç–µ—Ä–∞—Ç–æ—Ä
counter = CountToFive()
for number in counter:
    print(number)
```
